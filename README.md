Calculator Project

A **simple and stylish Calculator** built using **HTML** and **CSS**. This project demonstrates basic web development skills, including page layout with HTML and styling with CSS. The calculator provides a clean and interactive user interface for basic arithmetic operations.

Features
- Addition, subtraction, multiplication, and division
- Percentage (%) and power (^) operations
- Clear (AC) button to reset input
- Decimal input support
- Fully responsive and user-friendly interface
- Styled entirely with CSS, no JavaScript required

Technologies Used
HTML5: For the structure and layout of the calculator  
CSS3: For styling, design, and responsive layout
